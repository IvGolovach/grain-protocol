# RC Records

This directory stores release-candidate governance records:

- `SIGNOFFS/`: signed claim readiness/signoff records.
- `REVOCATIONS/`: signed RC revocation records.

The current repository state only carries templates plus historical imported RC
records. Add live signoff or revocation records when a new RC window is active.

Use templates:
- `spec/rc/SIGNOFFS/template.json`
- `spec/rc/REVOCATIONS/template.md`

Policy reference:
- `spec/RC-POLICY.md`
