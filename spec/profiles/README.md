# Profiles

Profiles are normative constraints for v0.1:

- `cbor-profile.md` — strict DAG-CBOR + baseline limits + strict mode
- `cose-profile.md` — narrow COSE_Sign1 signature container rules
- `e2e-profile.md` — E2E envelope, deterministic nonce, manifest resolution
- `qr-profile.md` — GR1 QR pipeline
