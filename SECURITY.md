# Security Policy

Grain is a protocol and conformance suite intended to run in adversarial environments.
We take security issues seriously.

## Reporting a Vulnerability

Please do not open public issues for suspected security vulnerabilities.

Instead, email: **security@grainprotocol.org** (replace with your real address/domain)
Include:
- a clear description of the issue and impact
- minimal reproduction steps
- affected files/sections (NES paragraphs, if applicable)
- any suggested fixes/mitigations

We will acknowledge receipt and coordinate a responsible disclosure timeline.

## Scope

This policy covers:
- protocol specification (NES/CDDL/profiles)
- conformance vectors and harness contract
- reference implementations under `core/` and `sdk/` (when present)

## Non-goals

Grain does not guarantee truthfulness of signed content.
Signatures attest authorship and integrity only.
